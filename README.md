# mito
Independent gaming platform prototype

MITO is an independent gaming platform prototype.

It combines:
- a console-style interface
- an online account system
- a web store
- a synced game library

## Features

- Python + Flask backend
- Pygame console interface
- User login system
- Real-time library sync
- Game launcher support

## Status

This is an early development version.

## Goal

The goal of MITO is to explore how a small-scale gaming platform could work and support indie developers.
